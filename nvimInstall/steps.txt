PASOS PARA LA INSTALACION DE NEOVIM ACTUALIZADO Y LAZYVIM

recomendable eliminar nvim o vim junto con todos los packages si ya se encuentra instalado 

1. https://github.com/neovim/neovim/releases/tag/stable ir a la seccion 'assets' y descargar el archivo nvim.appimage

2. Ubicarse en la carpeta donde se descargo nvim.appimage y ejecutar el siguiente comando: sudo apt-get install fuse

3. Ejecutar el siguiente comando: hmod u+x nvim.appimage && ./nvim.appimage

4. Mover el archivo a /usr/local/bin/nvim con el siguiente comando: sudo mv nvim.appimage /usr/local/bin/nvim

5. Clonar repositorio de Lazy vim: https://www.lazyvim.org/installation y seguir los pasos especificados ahi.

En caso de tener configuracion propias de nvim, antes de instalar lazyvim, hacer caso al comando: # required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}

De lo contrario haga caso omiso

7. Importante instalar el fd [find directory] y el ripgrep para el plugin integrado en lazy vim: "Telescope"

Para fd: sudo apt install fd-find
Para ripgrep: sudo apt install ripgrep

8. Instalar el compilador de lenguaje C, C# o C++ si no se tiene, ejecutando el siguiente cmando: sudo apt install g++
